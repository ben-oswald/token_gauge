# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),

## [0.1.0] - 2026-04-19

### Added
- Initial public release.
- Panel indicator showing quota usage for Z.ai and GitHub Copilot.
- Preferences window for enabling providers, ordering, refresh interval, and credential source (Kilo CLI auth.json or manual API key via GNOME Keyring).
- Low-quota desktop notifications.
